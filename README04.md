# react에서는 데이터를 변할수있는 값이며 값이 변할때마다 UI를 변경해 달라고 react에게 알려야한다.그럴때 사용하는것이 useState()이다.

```
const [ num, setNum ] = useState(0) 
이 데이터를 변할수 있는값 ==> num
num을 변하게 하는 함수 ==> setNum 함수를 이용하여 num을 변경한다.
useState(0)의 0의 num의 초기값
```
